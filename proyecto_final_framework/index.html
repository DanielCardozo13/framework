Introduccion

Componente principales

La hoja de estilo archivo_base.css en la carpeta DANCAR/base/ es el primer y más importante archivo de framework DANCAR. Este establece los estilos bases (por ejemplo, cuadrículas, formularios, etc.). Esta hoja de estilos es necesaria para todos los diseños basados ​​en el presente framework y no debe cambiarse.

Clases de tipo Grid

Las grids o cuadrículas están envueltas por un elemento con la clase .dc-grid. Para permitir un control independiente de los anchos de las grid o rejillas y columnas se utilizan las clases dc-g[xx] en donde se remplaza en los parentecis cerrados por valores de ancho y la clase .dc-grid_box que contiene margen, relleno o valores de borde..

Opciones de estilos para formularios

Hay un estilo predeterminado lineal (apilado, 60% ancho de elementos de formulario) y el estilo del 100% del ancho del contenedor con la clase dc-full.

clases CSS Descripción
por default El ancho de todos los elementos del formulario se acomodan al 60% del ancho del formulario.
.dc-full El ancho de todos los elementos de formulario se acomodan al 100% de ancho. Este estilo de formulario es especialmente útil para formularios pequeños.

Tipos de Botones

Los estilos de botones se pueden aplicar a cualquier elemento utilizando las siguientes clases.
